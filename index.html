Definiciones

* Css

Css es un lenguaje de diseño hecho para agragar estilos a paginas escritas en HTML o XML con el fin de hacerlas mas llamativas y mas comodas a la vista. Gracias a este lenguaje las paginas son menos cuadriculadas y el usuario puede entener mejor el proposito y opciones que la pagina ofrece.

* JavaScript

JavaScript es un lenguaje de alto nivel orientado a objetos y principalmente va vinculado con una pagina HTML para creaciones de paginas web. Esta es mayormente usada para un uso mas orientado a cosas complejas como lo son proyecciones de objetos 3D/2D y tambien es capaz de tener la pagina web constantemente actualizada.
